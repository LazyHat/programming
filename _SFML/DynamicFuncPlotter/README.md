# Dynamic_Func_Stack_Plotter

Эта программа строит график.

Функция задается динамически, с помощью стековых команд.
Регистр не учитывается.

Изначально на стек кладется x.

1. Унарные опрерации - применяются к последнему элементу стека:

   1. SQRT - замена числа на его квадратный корень
   2. COS - замена числа но его косинус
   3. SIN - замена числа на его синус
   4. TAN - замена числа на его тангенс
   5. CTAN - замена числа на его котангенс
   6. ASIN - замена числа на его арксинус
   7. ACOS - замена числа на его арккосинус
   8. ATAN - замена числа на его арктангенс
   9. ACTAN - замена числа на его арккотангенс

2. Бинарные операции - применяются к двум последним элементам стека, результат кладется на конец стека(аргументы удаляются из стека):

   1. ADD - Сложение
   2. SUB - Вычитание
   3. MUL - Умножение
   4. DIV - Деление

3. Операции со стеком:
   1. DUP - Положить на стек копию числа
   2. OVER - Положить на стек копию 2го элемента стека(предпоследний элемент)
   3. SWAP - Поменять местами два последних элемента стека

Используйте END для завершения написания функций и построения графика.
Используйте EXIT для выхода из программы.
